# Telematrix Pro

Описание проекта Telematrix Pro.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Структура проекта

```
telematrix-pro/
├── .cursorrules          # Правила для AI
├── requirements.txt      # Зависимости
├── main.py              # Точка входа
├── src/                 # Исходный код
│   ├── core/           # Ядро приложения
│   ├── ui/             # Пользовательский интерфейс
│   └── plugins/        # Плагины
└── README.md           # Документация
```
